# wless(WIP)

window less, wlroots based wayland compositor.

## Key bindings

- `Alt + Tab` 切换到下一个窗口，从近到远
- `Alt + Shfit + Tab` 同上但从远到近
- `Win + Tab` 切换到当前屏幕的下一个窗口，从近到远
- `Win + Shfit + Tab` 同上但从远到近
- `Win + Enter` 启动新终端 foot
- `Win + w` 关闭窗口
- `Win + Shift + Esc` 退出
